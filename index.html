📖   Setup Guide   📖

How to Customize This Website

Follow these steps to add your own photos, music, and go live!

1
Open the HTML file in a text editor
Right-click on happy-birthday-junlyn.html → Open With → Notepad (Windows) or TextEdit (Mac). You can also use VS Code (free download).
2
Upload your 3 photos somewhere online
Go to imgbb.com (free, no account needed). Click "Choose photo" → upload your photo → click "Upload" → copy the Direct link (ends in .jpg or .png).
Tip: The direct link looks like: https://i.ibb.co/abc123/photo.jpg — make sure to copy the one that ends in an image extension!
3
Find the photo section in the code
Press Ctrl+F (Windows) or Cmd+F (Mac) in Notepad. Search for this text:
const photoSrcs=[null,null,null]
4
Replace null with your photo links
Find that line near the bottom of the file (inside <script>) and replace null with your image URLs like this:
const photoSrcs=[ 'https://i.ibb.co/abc123/photo1.jpg', 'https://i.ibb.co/def456/photo2.jpg', 'https://i.ibb.co/ghi789/photo3.jpg' ];
Note: Each link must be inside single quotes ' ' and separated by commas. Keep the square brackets [ ].
5
Edit the photo captions (optional)
Search for photo-caption in the code. You'll find 3 of them. Change the text inside to your own captions, like "Our first trip 🌴" or "Best day ever 💕".
<div class="photo-caption">Your caption here ✨</div>
6
Save the file
Press Ctrl+S (Windows) or Cmd+S (Mac) to save. Then double-click the HTML file to open it in your browser and check your photos!
1
Choose your audio file
Pick an .mp3 file — a birthday song, her favorite song, or even a voice message you recorded. You can use any MP3 file on your computer.
Tip: Free birthday songs: search "happy birthday mp3 free download" on Google, or use sites like pixabay.com/music for royalty-free songs.
2
Upload your MP3 online
Go to archive.org (free, no limit). Create a free account → click "Upload" → upload your MP3 → after it uploads, right-click the MP3 file name → "Copy link address". OR use GitHub (see the "Go Live" tab — upload your MP3 there too).
3
Find the audio line in the code
Press Ctrl+F and search for:
YOUR_AUDIO_FILE.mp3
4
Replace with your audio URL
Replace YOUR_AUDIO_FILE.mp3 with the link to your MP3 file:
<source src="https://yourlink.com/song.mp3" type="audio/mpeg">
If hosting on GitHub Pages: Just put the mp3 file in the same folder as the HTML and write src="song.mp3" — no full URL needed!
5
Test it in your browser
Save the file and double-click to open in Chrome or Edge. Click the ▶ play button. If you hear sound — you're done! 🎉 If not, check that the URL is correct and ends in .mp3.
1
Create a free GitHub account
Go to github.com → click "Sign up" → create your free account with your email. GitHub Pages lets you host any website for free!
2
Create a new repository
After logging in, click the + icon (top right) → "New repository". Name it something like junlyn-birthday. Make sure it's set to Public. Click "Create repository".
3
Upload your files
Inside your new repository, click "uploading an existing file". Drag and drop happy-birthday-junlyn.html (and your MP3 if using one). Click "Commit changes".
Important: Rename your HTML file to index.html before uploading so it loads automatically as the homepage!
4
Enable GitHub Pages
In your repository, click Settings → scroll down to "Pages" in the left menu → under "Branch", select main → click Save. Wait 1-2 minutes.
5
Get your website link!
Your site will be live at:
https://YOUR-USERNAME.github.io/junlyn-birthday/
Example: If your GitHub username is maria123, your link is: https://maria123.github.io/junlyn-birthday/ — share this link or use it for the QR code!
1
Get your website link first
Complete the "Go Live" steps above to get your GitHub Pages URL. You need a live URL before making a QR code.
2
Open the QR code generator
Open the file birthday-qr.html (included in your download) in any browser. OR go to qr-code-generator.com for free online QR generation.
3
Paste your link and generate
Paste your website URL into the input box → click "Generate ✨" → your QR code will appear instantly. Click "Download QR Code" to save it as a PNG image.
4
Print or share the QR code
You can: (A) Print it and stick it inside a physical birthday card 🎂 — when Junlyn scans it with her phone camera, it opens the website! (B) Send the QR image on Messenger, Viber, or WhatsApp.
How to scan: Just open the Camera app on any iPhone or Android, point it at the QR code, and tap the notification that pops up. No special app needed!
5
Done! 🎉
Junlyn scans the QR → her phone opens the birthday website → she hears the music, sees the photos, reads your message, and plays the mini game. She'll love it! 💕
